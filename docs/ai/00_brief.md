# Brief

`collab-cli` is the command-line orchestrator for UxmalTech collaborative workflows with canonical architecture. It manages the full lifecycle: from initial onboarding of a repository through Docker infrastructure, AI provider configuration, canon synchronization, and domain generation.

## Ecosystem role

- `collab-cli` is the user-facing entry point that orchestrates everything
- `collab-architecture` is the framework-level canonical source of truth
- `collab-architecture-mcp` is the MCP server exposing canon as graph + vectors to AI agents
- Application canons (configurable repo) hold domain-specific rules generated by this CLI

## Core capabilities

- **Wizard onboarding** (`collab init`): 8-stage (file-only) or 14-stage (indexed) pipeline
- **Infrastructure management**: Docker Compose generation, Qdrant + NebulaGraph lifecycle
- **MCP server management**: Start/stop/status of the MCP runtime
- **Canon synchronization**: Download/update canonical architecture from GitHub
- **AI provider auto-detection**: Codex, Claude, Gemini, Copilot with MCP config generation
- **Domain generation**: `collab init --repo=<package>` analyzes a repo and generates domain files in the configured architecture repo
- **Diagnostics**: System health checks, compatibility verification, doctor command

## Key design principles

- Two modes of operation: `file-only` (no Docker) and `indexed` (Qdrant + NebulaGraph + MCP)
- Stage-based orchestrator with persistent state for recovery (`--resume`)
- Side-effect executor with global `--dry-run` support
- Ecosystem manifest for cross-repo version compatibility
- Provider-agnostic AI client with pluggable model registry

## Governance and delivery

- `CONTRIBUTING.md` enforces English-only issue policy and structured issue quality
- `.github/ISSUE_TEMPLATE/work-item.yml` standardizes Context/Problem/Scope/Acceptance/Dependencies
- `install.sh` supports deterministic latest-main install/update without sudo
- `docs/release.md` defines npm-first distribution, version pinning, and rollback flow
